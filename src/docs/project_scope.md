# Alcance del Proyecto: "Healthy Eating" 

## Descripción General
Healthy Eating es una aplicación web diseñada para personas que quieren llevar un registro sencillo de sus comidas diarias, bebidas y ejercicio realizado. Todo ello a traves de una interfaz amigable y facil de usar.

## Usuario Objetivo 
Personas entre 18 y 65 años que inician en el mundo de la vida saludable y necesitan una herramienta para llevar un registro de sus comidas, bebidas y ejercicio realizado.

## Funcionalidades Principales (Core Features) 
1. **Registro de Comidas:** El usuario puede agregar una comida, bebida y/o snack, especificar tipo (desayuno, almuerzo, cena, snack) y cantidad de calorías. 
2. **Control de Hidratación:** Un contador simple de vasos de agua consumidos en el día. 
3. **Dashboard:** Una vista principal que muestra el resumen del día (Total calorías consumidas, Total agua consumida, cantidad de comidas al dia ), 
4. **Historial:** Mostrar los registros de días anteriores.

## Reglas de Negocio 
- No se requiere registro de usuario en la primera versión (datos guardados en LocalStorage inicialmente). 
- La interfaz debe ser "Mobile First" (pensada para celulares).
